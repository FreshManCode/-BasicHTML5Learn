### 1.函数
> 函数就是将具有一定功能的一段JS代码的封装,可以在程序的多个地方被反复调用

* 1.定义函数
	* 格式1:
	
				function 函数名 ([形参列表]) {
					函数体:
					[return <表达式>;]
				}
	* 格式2:
			
			var 变量名 =  function 函数名 ([形参列表]) {
					函数体:
					[return <表达式>;]
			}		
	*  格式3:
			
			;(function([形参列表]) {
					函数体:
					[return <表达式>;]
			})(实参列表)
			//该函数为立即执行函数(IIFE),它自动调用自身,不被在其它地方被被调用,一般用于JS库或JS插件的封装或闭包处理。	
			//注意:为了避免与前面的代码拼接在一起,最好在该函数前加上一个分号;	
		  
	
		  
####  	1.2.函数调用
	*  函数名([实参列表]) ;
	*  Tips:
		* 函数不会自动执行(IIFE(立即执行函数)除外),必须通过调用才能执行。

#### 1.4.arguments 对象
> 返回实参列表的一个伪数组

* 一般用在不确定传过来的实参个数的情况下;	

### 2.对象(object)
> 对象,其实就是一种类型,即引用类型。用于将数据和功能组织在一起。

>对象由属性和方法组成,通常用键值对定义。 

* 1.对象定义
	* a)new 构建
		* new Object();
	* b) 字面量定义
	
			var obj = {
				key:value,
				fn:function() {
					...
				}
			}   
* 2.对象引用
	a:对象名.属性名 | 对象名.方法名([实参列表])
	b:对象名[属性名]
	
### 3. 数组(Array)
> 数组将一组数据组合到一起,并存入到一个变量中,数组是有序排列的,占用一段连续的内存空间;一个数组可以存储不同类型的数据。

* 1.定义数组
	* a. new Array([值列表]) 
	* b. 字面量创建
		* var array = [值列表] 

* 2.获取数组元素值
	* 数组名[下标] //下标可以是一个数值常量,也可以是一个表达式|变量
* 3.数组的遍历
	* 一维数组用一个循环实现;二维数组必须用双重循环实现。
	* 1.for循环
	* 2. for ... in 循环 (推荐)
		
				 for (let i in arry1) {
	       			 console.log(arry1[i])
	    		}
		
	* 3. forEach() 循环

			    arry1.forEach((value,index)=>{
			        console.log('value:',value,'index:',index)
			    })
* 4.数组的属性和方法
	* 1. length属性: 获取数组的长度
	* 2. 方法:
		* i) push() 
			 *	向数组中添加数据到数组最后 
		* ii) pop()
			*  删除数组最后一个元素
		* iii) unshift ()
			* 向数组头部添加数据 
		* iv) shit()
			* 删除数组第一个元素 
		* v) 	concat()
			* 将两个多多个数组组合成一个数组。 
		* vi) reverse()
			* 对数组进行倒序处理
		* vii) join()   
			* 将数组转换为字符串。	 
		* viii) splice
			* 删除, 
  
### 4.Function类型
#### 1. 函数内部属性
* 在函数内部,有两个特殊的对象:arguments和this
*  callee
	* arguments 是一个类数组对象,包含着传入函数中的所有参数,主要用途是保存函数参数;
	* 这个对象还有一个名叫callee的属性,该属性是一个指针,指向拥有这个arguments对象的函数
	* 可以通过arguments.callee调用函数自身,一般用于函数的递归调用。
	* Tips:函数自己调用自己叫函数的递归调用
* this:指针对象
	* 在全局中this指向的是window(在JS中没有global中这个全局对象,而JS的全局对象是Window)
	* 在函数中this指向的是这个函数执行所操作的当前对象
