### 1.内置对象
* 1.)global 对象
 * 在js中没有global对象,web浏览器将global作为window对象的一部分加以实现
	* 2.方法:
		* a. encodeURIComponent():  对unicode进行编码处理
		* b. decodeURICompoenent(): 对unicode进行解码处理

### 2.面向对象
##### 2.1 创建对象
* 1.常规创建
	* new
	* 字面量 
* 2.工厂模式	
	* 通过封装函数实现创建一批相似的对象。
	* 缺陷:无法知道创建的对象是哪一个对象的实例 
* 3.构造函数
	* 构造函数是用来创建一个类(ES5中没有类的概念,实际上这里的构造函数就是类);
	* 类是对象的一个抽象符号化表示(把相同或相似的一部分对象抽离出来就形成了一个类)
	* 对象是类的实例化(具体化)(赋予一定的属性和功能)

* 	a.创建构造函数(类)
	* 语法 
	
			function 类型([形参列表])  {
				this.属性 = 参数
				...
				this.方法名 = functio() {
					函数体
				}
			
			}
* 	b.通过构造函数实例化对象:
	* new 类名([实参列表])  		
	
* 	使用构造函数的方法和使用工厂模式的方法的不同之处:
	* 1.构造函数方法没有显示的创建对象(new Object())  
	* 2.直接将属性和方法赋值给this对象
	* 3.没有return语句
	
### 3.基本包装类型
* 在基本数据类型汇总有3个特殊的类型的存在:String,Number和Boolean
* 上面三个基本类型都有自己的包装对象,有相应的属性和方法。调用方法的过程是在后台发生的,所以我们称作为基本包装类型。
* 通俗的讲就是基本类型的数据都有一个包装他们的类,这些类都有自己的属性和方法,这些基本类型的数据都可以直接去调用这些属性和方法。
* 3.1 Boolean类型
	* 没有自己的属性和方法  
* 3.2 Number 类型
	* a.方法
		* toString
		* toLocalString
		* to Fixed()
* 3.3 String类型
	* a:属性 length
	* b:方法
		* charAt(n) 返回指定索引位置的字符
		* charCodeAt(n) 返回指定索引位置字符的unicode编码 
		* concat(str1...str2)	将字符串参数串联到调用该方法的字符串
		* slice(n,m) 返回字符串n到m之间位置的字符串
		* substr(n,m) 返回字符串n开始的m个字符串
		* substring(n,m) 同上
		* indexOf(str,n) :从n开始搜索第一个str,并将搜索的索引值返回
		* toLowerCase():将字符串转换为小写
		* toUpperCase():将字符串转换为大写
		* match(pattern):返回patter中的字串或null
		* repalce(pattern,replacement):用replacement替换pattern
		* search(pattern):返回字符串中pattern中开始位置
		* split(pattern): 返回字符串按指定pattern拆分的数组

### 4.变量,作用域及内存
#### 1.变量
* a.基本类型的变量
	* 基本类型变量值存储在栈中;通过变量名可以直接获取变量的值。
* b.引用类型的变量
	* 引用类型变量的值存储在堆中,在栈中存储的是引用类型的变量的地址(指针)  
	* 如果要获取引用类型变量的值,需要先从栈中获取地址,再按址查找,从而获取到值
	
#### 	2.作用域
* 	在ES5中作用域分为全局作用域和局部作用域两种;
*  在ES6中作用域分为全局作用域,局部作用域和块级作用域三种;
*  全局作用域:
	* 定义在函数外部的变量拥有全局作用域	  
*  局部作用域
	* 定义在函数内部的变量有用局部作用域
