### 6.JS的特点:
* 1.松散性
	* JS中的变量没有一个明确的类型,也叫弱类型的语言(允许将一块内存看作多种类型)。
* 2.对象属性
	* 对象的属性也可以映射为任意的数据。
* 3. 继承机制
	* JS是基于原型继承的	 	

* 7.使用javascript
	* 	1. 使用script标签 (只在当前文件有效)
		
				<script type="text/javascript" defer="defer" charset="utf-8">
			    	alert("test")
				</script>
			*  defer:表示所有DOM元素加载完成后加,再执行JS代码 (现在开发一般不需要)	
			*  charset:	字符编码(只要解决汉字乱码问题,现在开发一般不需要)
			*  **注意**: 
				*  script 标签可以卸载网页代码中的任意地方,因为JS是同步执行的,为了避免JS阻塞和操作DOM,最好写在body后
				*  如果要输出script标签时,必须要拆开写
						
						alert('</sc' + 'ript') ;
			
	* 2.在a标签中的href中写JS代码(实际开发中不建议这种使用,因为会影响性能)
			
			<a href="javascript:printLog('a标签中的js代码')">大家好</a>
			<a href="javascript:var a =10,b=20;alert('结果为:'+ (a+b))">加法计算</a> 	
			
	* 	3.使用script标签引入外部JS文件 (可以被多个HTML文件使用)
		
			<script src="js/01_test.js" type="text/javascript" async="async"></script>
			
	* 	说明:	
		*  1.src 引入的外部JS文件的路径和文件名(只能用于引入JS文件)
		*  2.async:异步加载JS代码(在加载DOM元素的同时可以运行JS代码)

### 	8.标识符
> 指变量,函数,属性的名字或者函数的参数

#####   标识符定义规则
* 1.第一字母必须是字母,下划线_或者$(美元符号)
* 2.其它字符可以是字母,下划线,美元符号或者数字
* 3.不能把关键字,保留字,true,false和null作为标识符
*  注意:
	* 在js中任意地方是严格区分字母大小写的 
	
	
### 9.注释 
* 1.单行注释
	* 一般用于对当前行的代码行的说明  //
	* 多行(块)注释,一般对下面的代码进行整体说明,且说明内容可能较多      /* 注释内容 */
	* 说明:
		* 1.注释语句在浏览器中不会被解析执行,仅起说明作用: 
		* 2.在项目中,要习惯去写注释,主要便于后期的项目维护。

### 10.常量,变量
* 常量:也叫直接量或字面量,在程序中直接给出具体的数据
	* 常量是不可变的
	* 如:100,true,'abc' 
* 变量:在内存中开辟一段空间用于存放某个数据。
	* 变量必须要有变量名,变量名必须遵从标识符的命名规范。 
	* 定义:
		* 1.只定义变量
		
				var x;
				var a,b,c  	
		* 2.定义变量且赋值
		
				var x1 = true;
				var y1 = 100,y2 = null; 
		
		*  说明:
			 1. 定义变量是不需要给出数据类型(松散式语言特点)
			 2. 变量可以重复定义,后面会覆盖前面变量
			 3. 不用var定义变量也可以,默认是为window对象添加了属性
			 
			 		name = '张三',相当于window.name = '张三'
			 4. 定义的变量如果没有赋值,系统将自动赋值为'undefined'
			 5. 一条语句可以以分号结束,也可以不加分号;如果是多条语句写在同一行上,必须要用分号隔开
			 6. 在书写代码时,除了字符串中可以使用中文标点之外,其它任意地方只能用英文标点符号。

### 12.数据类型
* 	数据类型指的是在内存存储的方式。
*  分为:
	* 1.基本数据类型
		* number:数值型
			* 用来表示一个数字,通常可用作进行加减乘除等运算
			* 分为:整型和浮点型(小数位可以浮动)
			* isNaN():用来判断是不是不是一个数
			* 100 :十进制
			* 0123:8进制
			* 0xae12 :16进制
		* string:字符型
			* 用引号(单/双) 引起来的一串字符
		* boolean:布尔(逻辑)型
			* true/false 
		* null:空
			* 表示一个空对象的地址指定指向的为空 
		* undefined:未定义 
			* 表示定义了一个变量,但如果没有给这个变量赋值,系统将自动赋值为undefined 
		 
	* 2.引用数据类型  
		* object:对象型
			* 用来申明或存储一个对象(对象,函数,正则,字符,数值等)  
				
					var a = new Number(10)
					
					var obj = {
					        name : '李白',
					        no :'007',
					  }
					  
### 13.运算符
* 1.算术运算符
	* + - * / % ++ --  
* 2.字符串运算符
	* +:用于实现字符串链接。 
* 3.关系运算符
 	* > < >= <= ==  != !== 
 	* 返回的结果只能是true/false
 	* 比较方法:
 		* 1.数值比较,是比较其大小 :
 		* 2.字符比较,是比较ASCII码值的大小
 		* 3.汉字比较,是比较其Unicode编码值的大小
 			* 可以通过charCodeAt()获取编码值
 		* 4.== 和!= 只比较值,不比较类型;===和!== 既比较值又比较类型
 		
 	* 比较:部分符号<数字<大写字母<小写字母<汉字  
 	 	 
* 4.逻辑运算符
	* && (与) ||(或) !(非)三种;返回的值一般是逻辑值true/false,也可能返回其它值
	* ! :(单目)一元运算符 
		* !true ->false , !false->true 
	* && :逻辑与(双目/二元)运算符
		* 只要有一个操作数为false,结果为false
		* 注意:
			* 如果两个中任意一个操作数非逻辑值,第一个操作数的结果为true时,返回第二个操作数的值
			* 如果第一个操作数为false,返回第一个操作数的值。
	* || :逻辑或(双目/二元)运算符
		* 只要有一个操作数为true,结果为true
		* 注意:
			* 如果两个中的任意一个操作数为非逻辑值,第一个操作数的结果为true时,返回第一个操作数的值;
			* 第一个操作数的结果为false时,返回第二个操作数的值; 
	* 短路运算:
		* 1.&& 运算时,如果第一个操作数为false时,不需要计算第二个操作数,结果返回false
		* 2.|| 运算时,如果第一个操作数为true,不需要计算第二个操作数,结果返回true    
* 5.位运算符
	* 很少用 
* 6.三目运算符  
	* 语法:
		* 表达式1 ? 表达式2 : 表达式3
		* Tips:
			* 如果表达式2或表达式3较为复杂,建议使用if语句或switch语句实现

### 14 流程控制
* JS是一门既面向过程,也是面向对象的解释性语言。
* 面向过程:按照代码书写的顺依次执行(OOP)
* JS也是一门结构性语言
* JS的结构分为顺序结构,分支(条件/选择)结构和循环结构三种
* 顺序结构:按照代码的书写顺序依次执行,一般包含初始化,赋值,输入/输出等语句。
* 条件结构:用if或switch语句实现,其中的代码是有条件选择执行的。
* 循环结构:某部分代码在制定的条件范围内反复执行,用for/for...in/forEach/while/do...while语句实现

### 15 各种循环的区别:
* i:for循环只能用于循环次数已知的情况;而while和do...while可以用在循环次数已知或未知的情况,一般循环次数已知用for较多;for...in用于遍历数组和对象;forEach()用于遍历数组,for循环也可以遍历数组,单性能较差。

* ii: break和continue语句
	* break语句可以用在switch和循环语句中(forEach循环除外),表示跳出(结束)情况或循环
	* continue语句只能用在循环语句(forEach循环除外)中,表示结束本次,继续下一次循环
	* 注意:
		* a. break和continue语句必须单独存在,后面不能添加其它代码
		* b. break和continue语句 一般放在if语句中
